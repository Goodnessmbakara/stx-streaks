# STX Streaks

**STX Streaks** is a decentralized application (dApp) built on the Stacks blockchain, designed to help users build consistent habits through on-chain accountability. It leverages Clarity smart contracts for logic and security, and integrates WalletConnect via Reown AppKit for a seamless user experience.

## Features

- **Habit Tracking**: Define and track your daily streaks.
- **On-Chain Accountability**: Commit to your streaks with STX transactions.
- **Wallet Connectivity**: Connect securely using WalletConnect supported wallets (e.g., Xverse, Leather).
- **Gamified Experience**: Earn rewards and build your reputation.

## Tech Stack

- **Frontend**: React + TypeScript + Vite
- **Blockchain**: Stacks (Clarity)
- **Wallet Integration**: Reown AppKit (WalletConnect)
- **Styling**: Vanilla CSS / Custom Design System

## Getting Started

### Prerequisites

- Node.js (v18+)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone git@github.com:Goodnessmbakara/stx-streaks.git
   cd stx-streaks
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

## Development

- **Build**: `npm run build`
- **Test**: `npm run test`
- **Lint**: `npm run lint`
